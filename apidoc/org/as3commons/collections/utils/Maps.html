clone()method
public static function clone(source:IMap, keyFilter:Function = null, itemFilter:Function = null):IMap

Clones the supplied IMap instance returning a new IMap of the same type. If filter are specified, the resulting map only contains mappings that meet the supplied predicates.

The key filter function accepts the current key and returns a boolean value (true if the key is accepted).

  function keyFilter(key :  : Boolean {
  var accept : Boolean = false;
  // test the key
  return accept;
  }
    
  var iterator : IIterator = new MapFilterIterator(map, keyFilter);

The item filter function accepts the current item and returns a boolean value (true if the item is accepted).

  function itemFilter(item :  : Boolean {
  var accept : Boolean = false;
  // test the item
  return accept;
  }
    
  var iterator : IIterator = new MapFilterIterator(map, keyFilter, itemFilter);
Parameters
source:IMap — the IMap instance to operate on.
 
keyFilter:Function (default = null) — Function which will be be applied to each key in the source map.
 
itemFilter:Function (default = null) — Function which will be be applied to each item in the source map.

Returns
IMap — A new IMap instance.

filterKeys()method 
public static function filterKeys(map:IMap, predicate:Function):IMap

Filters the supplied IMap instance returning a new IMap of the same type which only contains mappings where the key meets the supplied predicate.

Parameters

map:IMap — the IMap instance to operate on.
 
predicate:Function — Function which will be be applied to each key in the supplied map. This function should expect a single argument (the key) and return a Boolean value, true if the key should exist in the returned Map instance or false if it should not be included in the returned Map.

Returns
IMap — a new Map instance consisting only of mappings where the key met the supplied predicate.

itemOrAdd()method 
public static function itemOrAdd(map:IMap, key:*, item:*):*

Attempts to retrieve the item mapped to the supplied key. If the supplied key is not mapped then the supplied value will mapped to the key and returned.

Parameters
map:IMap — the IMap instance to operate on.
 
key:* — the key to perform the lookup using.
 
item:* — if no mapping exists for the supplied key a new mapping will be added between the key and this value.

Returns
* — if a prior mapping exists for the supplied key then the assosiated value in the map will be returned; if no prior mapping exists the supplied value will be returned instead.

itemOrValue()method 
public static function itemOrValue(map:IMap, key:*, defaultValue:*):*

Attempts to retrieve the item mapped to the supplied key. If the supplied key is not mapped then the supplied default value will be returned instead.

Parameters
map:IMap — the IMap instance to operate on.
 
key:* — the key to perform the lookup using.
 
defaultValue:* — the value to return if the supplied key does not exist in the supplied Map.

Returns
* — if the supplied key exists in the map then the value mapped to that key will be returned, otherwise the supplied defaultValue will be returned instead.
