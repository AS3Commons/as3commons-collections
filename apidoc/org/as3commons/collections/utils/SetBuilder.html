Class SetBuilderMethods
 
Packageorg.as3commons.collections.utils
Classpublic class SetBuilder

Uses the Builder Pattern to simplify the creation of new ISet instances in a fluent fashion.


Example
Creating a new LinkedSet instance using SetBuilder. <listing version="3.0"> const result : IMap = SetBuilder.linkedSet() .add("item-one") .add("item-two") .build(); // Creates the following Set: { item-one, item-two } trace(result); </listing>
