h2. AS3Commons Collections - ActionScript Collections Framework

The AS3Commons Collections project is the most serious AS3 collections resource in the web - providing both a sophisticated collections framework as well as a library of fast collection implementations.

h3. Features

* Small and consistent API
* Fast collection implementations
* Powerful iterators
* Collection util classes
* Data binding enabled collections
* 100% unit tested

h3. Implementations provided

*Collections*

* ArrayList, SortedList
* Set, LinkedSet, SortedSet
* Map, LinkedMap, SortedMap
* LinkedList, Treap

*Bindable Collections*

* ArrayListFx, SortedListFx
* SetFx, LinkedSetFx, SortedSetFx
* MapFx, LinkedMapFx, SortedMapFx

*Special purpose iterators*

* ArrayIterator, RecursiveIterator
* FilterIterator, RecursiveFilterIterator, CollectionFilterIterator, MapFilterIterator

*Utils*

* CollectionUtils, ListUtils, SetUtils, MapUtils, ArrayUtils
* ListBuilder, SetBuilder, MapBuilder
